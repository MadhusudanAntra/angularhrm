Services:->
   Services are used to transfer the data from one component to anothre component
   use services to connect the api and get the data from apis and then pass that to 
   controllers
   to connect with api/microservice use HttpClient service and use its methods
   get
   post
   put
   delete

   all above 4 methods will return observable

   HttpClient will be used for async communication.




   input/output properties :- (if the components are related one component must be parent and another must be child)
